## Instructions
- fork this repo 
- clone it (`git clone YOUR_REPO_URL`)
- make a new branch for your changes (`git switch -c YOUR_INITIALS/DESCRIPTIVE_ISH_NAME`)
- make a copy of `intros/TEMPLATE.md`, and rename it to `intros/YOUR_NAME.md`
- fill in your introduction!
- stage your changes (`git add .`)
- commit your changes (`git commit -m "DESCRIPTIVE COMMIT MESSAGE"`)
- push your changes (`git push`)
- create a PR (go back to the main repository, and there should be a button to do it?)
- boom!
